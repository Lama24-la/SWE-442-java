# Release Notes

## Version 1.0 - Initial Release

**Release Date:** 2025-11-28

### Features

- **String Reversal**: Reverse any given string
- **Vowel Counter**: Count the number of vowels in a string
- **Palindrome Checker**: Check if a string is a palindrome (ignores spaces and case)
- **Title Case Converter**: Convert strings to title case format

### What's Included

This initial release includes a fully functional Java utility tool with four string manipulation methods:

1. `reverse(String str)` - Returns the reversed version of the input string
2. `countVowels(String str)` - Returns the count of vowels in the string
3. `isPalindrome(String str)` - Checks if a string reads the same forwards and backwards
4. `toTitleCase(String str)` - Converts a string to title case

### Technical Details

- Language: Java
- No external dependencies required
- Compilation: `javac StringUtility.java`
- Execution: `java StringUtility`
